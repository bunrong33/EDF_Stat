# EDF_Stat

Ce projet contient des scripts Python pour le traitement, la préparation et l’analyse de données.

## 1. Présentation du projet

Chaque utilisateur doit donc :

1. cloner le dépôt 
2. créer un dossier `data` en local 
3. y placer ses propres jeux de données 
4. créer un environnement virtuel 
```
python -m venv .venv
```
Activer l'environnement virtuel:
Avec Powershell
```
.venv\Scripts\Activate
```

5. installer les dépendances avec `requirements.txt`.
```
pip install -r requirements.txt
```
